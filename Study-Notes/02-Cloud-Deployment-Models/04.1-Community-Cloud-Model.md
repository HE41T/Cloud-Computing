# Community Cloud Model (Data-Centric Perspective)

## Community Cloud คืออะไร
Community Cloud คือ Cloud ที่ออกแบบมาให้หลายองค์กร
ที่มีเป้าหมาย กฎหมาย หรือมาตรฐานเดียวกันใช้งานร่วมกัน

> เป็น Cloud สำหรับ \"กลุ่มองค์กรเฉพาะ\"

---

## ประเภทข้อมูลที่ใช้ใน Community Cloud
- Sensitive Data
- ข้อมูลที่ต้องแชร์ระหว่างองค์กร
- ข้อมูลที่ต้องมี Compliance เดียวกัน

---

## ตัวอย่างข้อมูล
- ข้อมูลผู้ป่วย (โรงพยาบาล)
- ข้อมูลนักศึกษา (สถาบันการศึกษา)
- ข้อมูลประชาชน (หน่วยงานรัฐ)
- ข้อมูลการวิจัยร่วมกัน

---

## ทำไมไม่ใช้ Public Cloud ทั่วไป
- Policy ด้านข้อมูลต้องเหมือนกัน
- มาตรฐานความปลอดภัยต้องตรงกัน
- Trust Model ระหว่างองค์กรสำคัญมาก

---

## ความสำคัญเชิงธุรกิจ
- แชร์ต้นทุน Infrastructure และ Security
- สร้างมาตรฐานกลาง
- เชื่อมโยงข้อมูลระหว่างองค์กรได้ง่าย

---

## ตัวอย่าง Community Cloud
- Government Cloud (G-Cloud)
- Health Cloud
- Education Cloud

---

## ประโยคอธิบายลูกค้า
> Community Cloud ช่วยให้องค์กรที่มีข้อกำหนดเหมือนกัน  
> ใช้ Cloud ร่วมกันได้อย่างปลอดภัยและมีมาตรฐาน
